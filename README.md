## Bem Vindo

Essa aplicação foi feita no intuito de fazer uma lista de contatos similar a dos celulares

### Funcionalidades

### Na mesma é possivel

- Adicionar contato
- Remover contato
- Visualizar um contato
- Ver lista de contatos

 ### Pesquisar por :
  - Nome,
  - Telefone,
  - Email
  - Entre outros

   ### Ordenar por :
  - Nome
