# Mobile Challenge Hausenn

## Bem Vindo

Essa aplicação foi feita para ser avaliado no desafio do link → https://github.com/HausennTechnologies/hausenn-mobile-challenge

### Funcionalidades

### Na mesma é possivel

- Adicionar contato
- Remover contato
- Visualizar um contato
- Ver lista de contatos

 ### Pesquisar por :
  - Nome,
  - Telefone,
  - Email
  - Entre outros

   ### Ordenar por :
  - Nome
